data_loader.py - The data management module¶

This module handles the fetching of the data from the local resources path, given in the configuration and arranging it for our purposes of estimations. For instance, the data for Example no. 1 can be fetched by:

get_data(ExperimentType.ExampleNo1) - Creating the data for Example no. 1 of the paper.

randomized_decompositions.pyx - Randomized algorithms module¶

This module contains the two randomized algorithms for matrix decompositions: the randomized SVD and randomized ID.

randomized_decompositions.random_id¶

The function for Random ID algorithm.

Parameters

  • A (GeneralMat) – The matrix to decompose.

  • k (const int) – Approximation rank.

  • increment (const int) – The extra sampled columns in the approximation (beyond the first k columns.

Returns

Matrices \(B, P\) for which the ID approximation is \(BP\)

randomized_decompositions.random_svd¶

The function for Random SVD algorithm.

Parameters

  • A (GeneralMat) – The matrix to decompose.

  • k (const int) – Approximation rank.

  • increment (const int) – The extra sampled columns in the approximation (beyond the first k columns).

Returns

Matrices \(U, V\) and \(\Sigma\) for which the SVD approximation is \(U\Sigma V^{T}\)

matrices_classes.pyx - Matrix representations module¶

This module contains classes for convenient representations of matrices.

class matrices_classes.ExperimentNo5Form¶

A class for storing a matrix \(A\in\mathbb{R}^{m \times n}\) as \(A=uv^{T}+\sigma I_n\) where \(u=(1,0,...,0)\) and \(v^{T}=\frac{1}{\sqrt n}(1, 1, ..., 0)\)

class matrices_classes.MatInIDForm¶

A class for storing an ID decomposition for a matrix \(A\in\mathbb{R}^{m \times n}\): \(A=BP\) where \(B\in\mathbb{R}^{m \times r}\) in the skeleton matrix and \(P\in\mathbb{R}^{r \times n}\) is the interpolatory matrix.

class matrices_classes.MatInSVDForm¶

A class for storing an SVD decomposition for a matrix \(A\in\mathbb{R}^{m \times n}\): \(A=U\Sigma V^{T}\) where \(U\in\mathbb{R}^{m \times r}\) and \(V\in\mathbb{R}^{r \times n}\) have orthonormal columns and \(\Sigma\in\mathbb{R}^{r \times r}\) is diagonal and positive-definite.

main.py - The main module of the project¶

This module contains the config for the experiment in the “config” function. Running this module invokes the main() function, which then performs the experiment and saves its results to the configured results folder. Example for running an experiment: python main.py

main.main()¶

The main function of this project

This functions performs the desired experiment according to the given configuration. The function runs the random_svd and random_id for every combination of data_size, approximation rank and increment given in the config and saves all the results to a csv file in the results folder (given in the configuration).
